## 第二章 导航--文件系统

**pwd**: 查看当前工作目录

**ls**: 列出目录

**cd**: 改变目录

绝对路径名: 从根目录开始,其后紧接着一个又一个树分支,直到到达目标目录或文件.

```
$ pwd
/home/terence
# 其中第一个"/"表示根目录/, "/home/terence"表示就是绝对路径,表示"根目录下的home文件夹下的terence"
```

绝对路径名:  从工作目录开始. 有一些特殊符号来表示文件系统树中的相对位置: 符号"."表示工作目录, 符号".."表示工作目录的父目录.

**Linux文件命名中不要嵌入空格,以免命令行出现错误**
