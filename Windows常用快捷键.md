# Windows常用快捷键



Win+E 	打开Windows的资源管理器

Win+M	最小化所有被打开的窗口

Win+D 	快速显示桌面

Win+Tab	在任务栏上的按钮间循环

Win+← \ →	分屏操作\

Win+↓\↑	窗口最小化\最大化切换

Win+空格	切换多个输入法

Win+数字键	打开**任务栏**上第n个程序

Ctrl+Shift+N	快速创建文件夹

F3 	搜索文件或文件夹

Shift+Del 	永久删除所选项，而不将它放到“回收站”中

**Alt+Tab	快速切换窗口**

**Ctrl+Shift+T	恢复被误删掉的网页**



Crl + A 全选
Ctrl + X 剪切
Ctrl + Z 撤销
Ctrl + S 保存
